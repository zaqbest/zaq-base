#### bucket创建与上传数据（需要加成minio已经启动完成）

```
# 添加minio的host
mc config host add cyl-lp http://10.30.3.81:30912 admin root123456

#创建idsp的bucket
mc mb cyl-lp/idsp

#修改idsp为public
mc anonymous set public cyl-lp/idsp

# 上传文件
mc cp --recursive  miniodata/public  cyl-lp/idsp

#查看上传的文件
mc tree -f cyl-lp/idsp
```
